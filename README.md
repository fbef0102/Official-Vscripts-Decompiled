1. Download [L4D2 VScript Editor](https://www.mediafire.com/?q4dqbs554p494vk)
2. Open CMD -> cd "L4D2VScriptEditorBeta"
3. ```.\vice3 -d -k SDhfi878 -x .nut xxxx```
